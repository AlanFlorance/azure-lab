# Azure Lab Deployment

Currently this deploys 3 main compomnents

1. Keyvault
   - Resource Group
     - Secrets

2. Identity
   - Resource Group
     - Domain Controller Virtual Machines
     - Availability Set
     - Virtual network
     - Domain Creation
     - Bastion subnet
3. Connectivity
   - Resource Group
     - Virtual network
     - Virtual network gateway
     - Local network gateway
     - Virtual network peerings
